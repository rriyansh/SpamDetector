# Spam Email Detector

A simple Machine Learning project that detects whether an email message is **Spam or Not Spam**.<br>
This project uses **Natural Language Processing (NLP)** and a **Naive Bayes classifier** to analyze email text and classify it.<br><br>

This is a beginner-friendly project created to understand how machine learning models work for text classification.

---

## Features

✔ Detects spam emails using Machine Learning<br>
✔ Uses text vectorization to convert messages into numerical data<br>
✔ Classifies email as **Spam 🚨** or **Not Spam ✅**<br>
✔ Beginner-friendly project for learning NLP basics


##  Machine Learning Concepts Used

• Text Vectorization<br>
• Bag of Words Model<br>
• Naive Bayes Algorithm<br>
• Train/Test Data Split


##  Tech Stack

Python 3<br>
Pandas<br>
Scikit-learn<br>
